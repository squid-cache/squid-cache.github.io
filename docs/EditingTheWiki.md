# Editing the Squid Wiki

This wiki is hosted on [GitHub pages](https://docs.github.com/en/pages).
Its contents are in the repository [squid-cache/squid-cache.github.io](https://github.com/squid-cache/squid-cache.github.io).

In order to contribute to it, the easiest way is to clone the repository, edit it, and submit any proposal for changes as a PR.

Thanks for any contributions!
